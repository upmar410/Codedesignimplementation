
# Code Design Implementation

This is a code bundle for Code Design Implementation. The original project is available at https://www.figma.com/design/d1fK6ppoWjqi2UoS96FsKr/Code-Design-Implementation.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Deployment

This app is configured for GitHub Pages deployment. The app will automatically deploy when you push to the `main` branch.

### Live Site
Once GitHub Pages is enabled, your app will be available at:
`https://upmar410.github.io/Codedesignimplementation/`

### Enabling GitHub Pages

1. Go to your repository settings on GitHub
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "GitHub Actions"
4. The workflow will automatically build and deploy your app on every push to `main`

### Manual Build

To build the app locally:
```bash
npm run build
```

The built files will be in the `build` directory.
